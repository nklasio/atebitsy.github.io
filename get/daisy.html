Daisy

A better audio interface experience.

Daisy adds the current audio route icon to the volume slider in addition to a route label in expanded mode.

Daisy supports all audio outputs with a default icon as well as different icons for the following devices:

  1. AirPods
  2. Apple TV
  3. Beats Solo
  4. Beats Studio
  5. Beats X
  6. HomePod
  7. Powerbeats

The following non-Apple devices are also include support for icons in the audio route picker and battery widget:

  1. CarPlay
  2. Bose QC35
  3. Bose SoundSport Pulse

Also included is an option to show volume percent in the slider.

Supports 13HUD by @Muirey03
